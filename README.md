
### Vue 3 + TypeScript + Vite


### 功能：
- 支持皮肤切换
- i18n UI框架中英文切换
- 全屏放大缩小
- 按需引入echart & element
- 工程化自动引入组件，无需手动引入
- 路由切换动画
- 业务组件基础封装
- 环境区分和打包
- 浏览器样式工程化兼容

### 技术栈
- vue@3.2.25
- vue-router@4.0.16
- pinia@2.0.14
- axios@1.4.0,
- element-plus@2.2.4
- typescript@4.5.4
- vite@2.9.9
- sass@1.52.2

### 启动项目
- 使用 `pnpm install` 依赖安装
- 使用 `npm run dev` 快速启动项目 

### eslint 一键修复
- 使用 `npm run eslint` 启用 `eslint` 自动修复

### prettier 一键格式化
- 使用 `npm run prettier` 启用 `prettier` 自动格式化

### 代码提交
- 使用 `git add .` 将修改的文件添加到暂存区
- 使用 `npm run commit` 将暂存区的文件进行提交
- 使用 `git push` 将提交内容推送到远程
